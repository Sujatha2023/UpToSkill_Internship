# 🚀 Modern Animated Login Page

A modern, responsive, and interactive login page built using **HTML5**, **CSS3**, and **JavaScript**. This project features a sleek glassmorphism design, smooth animations, and an engaging user experience suitable for portfolios and front-end development practice.

## 📌 Project Overview

This project demonstrates how to create an attractive login interface using modern web technologies. It focuses on responsive design, UI/UX principles, and interactive elements to provide a professional-looking authentication page.

## ✨ Features

* 🌈 Animated gradient background
* 💎 Glassmorphism login card
* 📝 Floating input labels
* 👤 Username and password fields
* 👁️ Show/Hide password functionality
* ⚡ Login button with loading animation
* ✅ Basic form validation
* 🎨 Hover effects and smooth transitions
* 🌐 Responsive design for desktop and mobile
* 🔗 Social media login icons (UI only)

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Google Fonts (Poppins)
* Font Awesome Icons

## 🚀 Getting Started

1. Clone the repository:

```bash

```

2. Navigate to the project folder:

```bash
cd UpToSkill_Internship/Modern_Login_Page
```

3. Open `index.html` in your browser.

Or simply use the **Live Server** extension in Visual Studio Code for the best development experience.
